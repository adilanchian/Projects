This program is a high level simulator of a tiny machine in regards to Computer
Architecture.

In order to use this program please do the following steps:

1. Through command line use the command: 'gcc ProgrammingProject3.c'
  - This will output an 'a.out' file
2. Verify that you have your test file in the same directory as this file
3. Run the program with the text file using command: './a.out nameOfTestFile.txt'
4. Follow the prompt and input data when prompted
